# 입시 인사이트 — 공공데이터 입시 분석기 (라이브 데모)

대입정보포털 「어디가」가 다루지 않는 **교차분석·시계열·진로연계**에 특화한 공공데이터 입시 분석 도구입니다. (전체 버전은 비공개 저장소)

## 데모 바로가기
👉 **https://aijhmin1031-eng.github.io/ipsi-insight-demo/**

## 포함 기능
- **대학 지표 분석**(플래그십) — 신입생 충원율·등록률을 **지역·설립(국공립/사립) 교차분석** + 지역별 평균 차트 + 충원율 낮은순 게시판. 어디가가 제공하지 않는 구조 분석.
- **전문대 합격선** — 학과별 합격자 평균/최저 성적 (3개년, 38,491건)
  - **합격 진단** — 내 학생부 등급 → 합격 가능 학과(안정/적정/소신)
  - **합격선 추세** — 연도별 합격선 변화(기회/주의)
- **학과**(6만) · **대학별 경쟁률**(2024·2025) — 게시판 + 실시간 필터

> 백엔드 없이 데이터(JSON)를 브라우저에 내장해 구동합니다(최초 로드 약 16MB). 검색창은 입력 즉시 필터, 목록은 페이지네이션됩니다.
>
> ⚠️ 솔직한 한계: 4년제 학과별 합격선·성적분석·모의지원은 어디가의 제도적 비공개 데이터라 제공하지 않습니다. 본 도구는 *대체*가 아니라 어디가가 약한 **데이터 분석** 영역에 특화합니다.

## 데이터 출처
공공데이터포털(**한국대학교육협의회**) 오픈API · **전문대학포털**(한국전문대학교육협의회) 공개 입시결과. 공공누리/이용약관에 따라 출처표시 후 활용.

※ 4년제 학과별 합격선은 대입정보포털 「어디가」의 비공개(대학 의무 제출) 데이터라 공공 수집이 불가하여 미포함. 전문대 합격선은 공식 포털 공개분으로 확보.
