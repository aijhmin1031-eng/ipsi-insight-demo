## 포함 기능 (핵심만)
- **통합검색** — 대학 + 학과 (학과 6만)
- **대학별 경쟁률** — 정원내 신입생 경쟁률 (2024·2025)
- **전문대 합격선** — 학과별 합격자 평균/최저 성적 (3개년, 38,491건)
  - **합격 진단** — 내 학생부 등급 → 합격 가능 학과(안정/적정/소신)
  - **합격선 추세** — 연도별 합격선 변화(기회/주의)

> 백엔드 없이 데이터(JSON)를 브라우저에 내장해 구동합니다(최초 로드 약 16MB). 전체 버전은 13개 뷰 + 자동 수집 파이프라인 + 분석 지표(선발성지수·추세신호·진출역검색 등)를 포함합니다.
